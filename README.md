# Nexus Recon Platform

A scalable OSINT and reconnaissance platform designed for security research, intelligence gathering, and professional reporting.

---

## Overview

Nexus Recon Platform helps security researchers collect, analyze, and visualize reconnaissance data through a modern web interface and automated reporting engine.

The platform combines multiple intelligence sources into a single dashboard and generates professional PDF reports for security assessments.

---

## Key Features

### Intelligence Collection

* Email Security Intelligence
* GeoIP Intelligence
* ASN Intelligence
* HTTPX Integration
* Domain Intelligence

### Reconnaissance Dashboard

* Modern UI built with Next.js
* Real-time scan tracking
* Findings management
* Risk scoring system

### Reporting

* Executive Summary
* Risk Distribution Charts
* Professional PDF Reports
* Contact Information Section
* Report Export Functionality

## Installation

### Clone Repository

```bash
git clone https://github.com/Hussainsaeed/scalable-recon-platform.git
cd scalable-recon-platform
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```txt
http://localhost:5000
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```txt
http://localhost:3000
```
